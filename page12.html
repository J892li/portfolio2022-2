Andrometa

Image Description

FBS but looks so sweet?

That is correct. Andrometa is a  robot shooting game. Players need to find the exit of the room in a limited time, collect enough candy and then shoot the enemy to get more points!

core mechanics

How the game work 

The core mechanics of this game are shooting, dodging enemies, leveling up after shooting enemies, and continuing the game.
